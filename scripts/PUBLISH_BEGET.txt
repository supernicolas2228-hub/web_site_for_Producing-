Выкладка на sanchaevkirill.ru (Beget, статика)

1) Пересобрать и получить архив (в корне проекта):
   npm run beget:zip
   Появится файл beget-upload-ДАТА_ВРЕМЯ.zip

2) Панель Beget → Файловый менедержер → каталог public_html домена sanchaevkirill.ru
   — загрузите zip и распакуйте «в эту папку» с заменой файлов,
   ИЛИ загрузите ВСЁ содержимое папки out/ (не саму папку out, а её файлы: index.html, _next, images, …).

3) Откройте сайт с жёстким обновлением: Ctrl+F5
   Проверка: внизу страницы строка «Сборка: …» с датой сборки.
   Прелоадер при новой вкладке или: https://sanchaevkirill.ru/?intro=1

4) Если сборка «не меняется» — закройте Cursor/проводник, снова npm run beget:zip
   (скрипт удаляет старые out/ и .next/).

5) Авто по SSH (если настроен хост beget в ~/.ssh/config):
   npm run deploy:beget
